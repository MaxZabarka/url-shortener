
# URL Shortener

A server side rendered URL shortener

## Demo

https://zabarka-url-shortener.herokuapp.com/
## Tech Stack

- Node.js
- EJS
- Express
- SQLite3


## Run Locally

Clone the project

```bash
  git clone https://github.com/MaxZabarka/url-shortener
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
